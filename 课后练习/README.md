## 使用方式

1. 每节课结束后，完成对应的课后练习
2. 练习基于 `demo/` 项目，在其基础上扩展
3. 每个练习都有明确的验收标准
4. 遇到问题可以用 AI 工具辅助，但要理解 AI 生成的代码
