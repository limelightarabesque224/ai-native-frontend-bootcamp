# 第 2 课练习：shadcn/ui

## 练习 1：添加组件（基础）

**目标**：使用 shadcn/ui CLI 添加组件并使用。

**任务**：

1. 在 `demo/` 项目中添加以下组件：
   ```bash
   npx shadcn@latest add button
   npx shadcn@latest add card
   npx shadcn@latest add badge
   ```

2. 创建一个新页面 `/demos/components`，展示这些组件的不同变体：
   - Button: default, destructive, outline, ghost, link
   - Card: 带 header、content、footer
   - Badge: default, secondary, destructive, outline

**验收标准**：
- [ ] 组件源码在 `components/ui/` 目录
- [ ] 页面展示所有变体
- [ ] 每个变体有代码示例

---

## 练习 2：定制组件（进阶）

**目标**：修改 shadcn/ui 组件源码，添加自定义变体。

**任务**：

修改 `components/ui/button.tsx`，添加一个新的 `gradient` 变体：

```tsx
gradient: "bg-gradient-to-r from-purple-500 to-pink-500 text-white hover:from-purple-600 hover:to-pink-600"
```

然后创建一个 "升级到 Pro" 的 CTA 按钮，使用这个渐变效果。

**验收标准**：
- [ ] 修改了 `buttonVariants` 的 `variant` 类型
- [ ] 添加了 `gradient` 样式
- [ ] 创建了使用示例
- [ ] TypeScript 类型正确

---

## 练习 3：构建复合组件（挑战）

**目标**：用 shadcn/ui 组件组合出一个复杂的业务组件。

**任务**：

创建一个 "用户设置面板" 组件，包含：
- Tabs（个人信息、安全设置、通知偏好）
- Form（React Hook Form + Zod）
- Switch（开关选项）
- Select（下拉选择）
- Button（保存、取消）
- Toast（保存成功提示）

需要的 shadcn/ui 组件：
```bash
npx shadcn@latest add tabs form switch select toast
```

**验收标准**：
- [ ] 使用至少 5 个 shadcn/ui 组件
- [ ] 表单验证正常工作
- [ ] 保存后显示 Toast 提示
- [ ] 所有组件样式统一
- [ ] 完全响应式
