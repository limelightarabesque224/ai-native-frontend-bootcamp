# 第 1 课练习：Tailwind CSS v4

## 练习 1：CSS 迁移（基础）

**目标**：将一个 CSS Modules 组件迁移到 Tailwind。

**任务**：

将以下 CSS Modules 代码改写为 Tailwind：

```css
/* ProductCard.module.css */
.card {
  display: flex;
  flex-direction: column;
  padding: 24px;
  background: white;
  border-radius: 12px;
  box-shadow: 0 2px 8px rgba(0,0,0,0.1);
  transition: box-shadow 0.2s;
}
.card:hover {
  box-shadow: 0 4px 16px rgba(0,0,0,0.15);
}
.image {
  width: 100%;
  height: 200px;
  object-fit: cover;
  border-radius: 8px;
}
.title {
  font-size: 18px;
  font-weight: 600;
  margin-top: 16px;
}
.price {
  font-size: 24px;
  font-weight: 700;
  color: #3b82f6;
  margin-top: 8px;
}
.button {
  margin-top: 16px;
  padding: 8px 16px;
  background: #3b82f6;
  color: white;
  border-radius: 8px;
  border: none;
  cursor: pointer;
}
```

**验收标准**：
- [ ] 用 Tailwind utility classes 替代所有 CSS
- [ ] 包含 hover 效果
- [ ] 只有一个文件（无独立 CSS）
- [ ] 视觉效果与原版一致

## 练习 2：响应式布局（进阶）

**目标**：用 Tailwind 构建一个响应式产品列表页。

**任务**：

创建一个产品列表页面：
- 移动端：单列布局
- 平板（md）：双列布局
- 桌面（lg）：三列布局
- 超宽屏（xl）：四列布局
- 包含筛选栏（移动端折叠，桌面端侧边栏）

**验收标准**：
- [ ] 使用 Tailwind 响应式断点（`sm:` `md:` `lg:` `xl:`）
- [ ] 使用 CSS Grid 或 Flexbox
- [ ] 筛选栏在移动端可展开/收起
- [ ] 不写任何自定义 CSS

## 练习 3：Tailwind v4 主题定制（挑战）

**目标**：使用 Tailwind v4 的 CSS-first 配置创建自定义主题。

**任务**：

在 `globals.css` 中使用 `@theme` 定义一套品牌主题：

```css
@theme {
  /* 定义品牌色 */
  /* 定义间距系统 */
  /* 定义圆角 */
  /* 定义字体 */
}
```

然后用这套主题创建一个完整的营销落地页，包含：
- Hero 区域（大标题 + 副标题 + CTA 按钮）
- 功能特性区域（3-4 个特性卡片）
- 定价区域
- Footer

**验收标准**：
- [ ] 使用 `@theme` 定义至少 5 个自定义变量
- [ ] 所有颜色使用自定义主题色
- [ ] 页面完全响应式
- [ ] 用 AI 工具辅助生成，记录 Prompt
