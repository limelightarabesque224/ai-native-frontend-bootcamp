# 第 6 课练习：Monorepo

## 练习 1：搭建 Monorepo（基础）

**目标**：使用 Turborepo + pnpm 搭建一个 Monorepo 项目。

**任务**：

创建一个 Monorepo 项目结构：

```
my-monorepo/
├── apps/
│   ├── web/          # Next.js 应用
│   └── docs/         # 文档站点
├── packages/
│   ├── ui/           # 共享 UI 组件
│   ├── utils/        # 工具函数
│   └── tsconfig/     # 共享 TS 配置
├── turbo.json
└── package.json
```

使用命令：
```bash
npx create-turbo@latest
```

**验收标准**：
- [ ] Monorepo 结构正确
- [ ] pnpm workspace 配置正确
- [ ] Turborepo 缓存正常工作
- [ ] 可以运行 `pnpm build` 构建所有包

---

## 练习 2：创建共享组件库（进阶）

**目标**：在 Monorepo 中创建一个共享的 UI 组件库。

**任务**：

在 `packages/ui/` 中创建一个组件库：

1. 初始化 shadcn/ui
2. 添加 5 个常用组件（Button, Card, Input, Badge, Avatar）
3. 在 `apps/web/` 中引用这些组件
4. 配置 Turborepo 的依赖关系

`turbo.json` 配置：
```json
{
  "pipeline": {
    "build": {
      "dependsOn": ["^build"],
      "outputs": ["dist/**"]
    }
  }
}
```

**验收标准**：
- [ ] `packages/ui` 可以独立构建
- [ ] `apps/web` 可以导入 `@repo/ui` 组件
- [ ] 修改组件后，依赖的 app 自动重新构建
- [ ] 使用 Turborepo 缓存加速构建

## 练习 3：AGENTS.md 多包管理（挑战）

**目标**：为 Monorepo 编写完整的 AGENTS.md 指南。

**任务**：

创建一个 Monorepo 级别的 AGENTS.md，包含：

1. **项目结构说明**
   - 每个 app 的用途
   - 每个 package 的职责
   - 依赖关系图

2. **开发规范**
   - 如何添加新的 app
   - 如何添加新的 package
   - 如何在 app 中引用 package
   - 版本管理策略

3. **AI 协作指南**
   - 修改共享组件时的注意事项
   - 如何处理跨包的重构
   - 如何避免循环依赖

4. **常用命令**
   ```bash
   pnpm add <pkg> --filter <workspace>
   pnpm build --filter=web
   turbo run build --force
   ```

**验收标准**：
- [ ] AGENTS.md 覆盖所有关键场景
- [ ] 包含实际的代码示例
- [ ] 用 AI 工具测试指南的有效性
- [ ] 记录 AI 理解 Monorepo 结构的准确度
