# 第 9 课练习：Vercel AI SDK

## 练习 1：实现真实 AI 聊天（基础）

**目标**：将 `demo/` 中的模拟聊天改为真实的 AI 聊天。

**任务**：

1. 获取 OpenAI API Key（或使用其他模型）
2. 创建 `.env.local`：
   ```
   OPENAI_API_KEY=sk-...
   ```

3. 实现 `/api/chat/route.ts`：
   ```ts
   import { streamText } from 'ai'
   import { openai } from '@ai-sdk/openai'

   export async function POST(req: Request) {
     const { messages } = await req.json()
     const result = streamText({
       model: openai('gpt-4o'),
       messages,
     })
     return result.toDataStreamResponse()
   }
   ```

4. 在前端使用 `useChat` Hook

**验收标准**：
- [ ] 真实 AI 响应正常
- [ ] 流式输出正常（打字机效果）
- [ ] 错误处理完善
- [ ] 加载状态显示

## 练习 2：实现 AI 工具调用（进阶）

**目标**：让 AI 可以调用工具（Function Calling）。

**任务**：

为 AI 聊天添加工具调用能力：

1. **天气查询工具**
   ```ts
   const tools = {
     getWeather: {
       description: '查询指定城市的天气',
       parameters: z.object({
         city: z.string().describe('城市名称'),
       }),
       execute: async ({ city }) => {
         // 调用天气 API
         return { city, temp: 25, condition: '晴' }
       },
     },
   }
   ```

2. **搜索工具**
   - 调用搜索 API
   - 返回搜索结果

3. 在 `streamText` 中配置工具：
   ```ts
   const result = streamText({
     model: openai('gpt-4o'),
     messages,
     tools,
   })
   ```

**验收标准**：
- [ ] AI 可以识别何时调用工具
- [ ] 工具调用参数正确
- [ ] 工具结果正确返回给 AI
- [ ] 前端显示工具调用过程

## 练习 3：构建 AI 搜索引擎（挑战）

**目标**：用 Vercel AI SDK 构建一个类似 Perplexity 的 AI 搜索引擎。

**任务**：

实现一个 AI 搜索应用：

1. **搜索流程**
   - 用户输入问题
   - AI 生成搜索查询
   - 调用搜索 API（Tavily/Serper）
   - AI 基于搜索结果生成答案
   - 显示引用来源

2. **技术实现**
   ```ts
   const tools = {
     search: {
       description: '搜索互联网获取最新信息',
       parameters: z.object({
         query: z.string(),
       }),
       execute: async ({ query }) => {
         // 调用 Tavily API
         const results = await tavily.search(query)
         return results
       },
     },
   }
   ```

3. **UI 要求**
   - 显示搜索过程（"正在搜索..."）
   - 显示引用来源（带链接）
   - 支持追问
   - 搜索历史

**验收标准**：
- [ ] 完整的搜索流程
- [ ] 引用来源可点击
- [ ] 答案质量高（基于真实搜索结果）
- [ ] UI/UX 流畅
- [ ] 支持流式输出
