# 第 7 课练习：MCP Tools

## 练习 1：Playwright MCP 基础（基础）

**目标**：使用 Playwright MCP 进行浏览器自动化。

**任务**：

安装并配置 Playwright MCP：

```bash
npx @playwright/mcp install
```

编写一个自动化脚本：
1. 打开 https://github.com
2. 搜索 "shadcn/ui"
3. 点击第一个结果
4. 截图保存

使用 Claude Code 的 MCP 集成执行这个脚本。

**验收标准**：
- [ ] Playwright MCP 配置正确
- [ ] 脚本可以成功执行
- [ ] 截图保存到本地
- [ ] 记录执行日志

## 练习 2：自动化测试（进阶）

**目标**：用 Playwright MCP 为 `demo/` 项目编写 E2E 测试。

**任务**：

为以下页面编写 E2E 测试：

1. **首页导航测试**
   - 验证所有课程卡片存在
   - 点击卡片跳转到对应页面

2. **表单测试**
   - 填写表单
   - 触发验证错误
   - 提交成功

3. **AI 聊天测试**
   - 输入消息
   - 验证 AI 响应
   - 验证消息历史

**验收标准**：
- [ ] 至少 3 个测试用例
- [ ] 所有测试通过
- [ ] 使用 Playwright 的最佳实践
- [ ] 测试可以在 CI 中运行
